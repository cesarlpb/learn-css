Selectores

Tipos principales de selectores:

  • Etiquetas
  • Clases
  • Ids
  • Otros

Etiquetas

Todas las etiquetas de HTML se pueden usar como selectores:

    
      p {
        color: red;
        text-align: center;
      }

Clases

El selector de clase empieza por por punto: .clase {...}:

    
      .clase {
        color: red;
        text-align: center;
      }

Ids

El selector de id empieza por por #: #id {...}:

    
      #id {
        color: red;
        text-align: center;
      }

Otros

Varias clases

Para seleccionar varias clases o elementos se separa con comas ,:

    
      p, h1, .clase {
        color: red;
        text-align: center;
      }

Parent-child

Para seleccionar child directo (elemento hijo) de otro elemento (parent) se usa >:

    
      /* Selecciona el <p> dentro de un <div> */
      div > p {
        color: red;
        text-align: center;
      }

      HTML:

      <div>
        <p>
          ...
        </p>
      </div>

Etiqueta con clase

Para seleccionar una etiqueta de una clase determinada se coloca el selector de clase seguido de la etiqueta:

    
      /* Selecciona los <p> de la clase "clase": */
      p.clase {
        color: red;
        text-align: center;
      }
